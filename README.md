Conversor de GLB a GLTF
Coloca el archivo .glb que deseas convertir en la carpeta glb-to-gltf con el nombre input.glb.

Abre la terminal, navega a la carpeta del proyecto:



cd "nombre-de-la-carpeta"
Ejecuta el script de conversión:


node convert.js
El archivo .gltf resultante se generará en la misma carpeta.
